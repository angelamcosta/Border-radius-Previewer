Border-radius Previewer

Border-radius Previewer is an application created to solve the Border-radius Previewer challenge proposed by the App Ideas Collection repository. The application shows a box which has a border-radius property applied to it. You can change the 4 border-radius values that are applied to the box (top-left, top-right, bottom-left, bottom-right). Once you reach the desired settings, you can copy properties to apply to your own projects.
